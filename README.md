## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000)

### Build

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Tech Stack

- React 18
- CSS3 with custom properties
- Canvas API for animations
- Modern ES6+ JavaScript
